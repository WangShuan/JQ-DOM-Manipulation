prop方法

對於布爾屬性的 比如說單選框勾選框等等
若沒有設定checked屬性 獲取到的值就會是undefined
故不能使用attr方法 必須改用prop方法
其用法與attr用法一樣

//checkbox、radio和select 返回的都是布爾值

* 1.使用prop()函式返回的結果:
console.log($("#chk1").prop("checked")); // false
console.log($("#chk2").prop("checked")); // true
* 2.使用attr()函式返回的結果：
console.log($("#chk1").attr("checked")); // undefined
console.log($("#chk2").attr("checked")); // checked

JQuery 動畫

三組基本動畫： show/hide slideDown/slideUp fadeIn/fadeOut
若是無法滿足自己想要的需求 還有自定義動畫可以編寫

//JQuery的用法都是大同小異的 類似的行為其操作模式都一模一樣

* 1.show/hide方法 不傳參數就沒有動畫
可以傳任意數字(單位毫秒) 或 三種字符串(fast/normal/slow)
fast:200ms / normal:400ms / slow:600ms
若是傳入任意的字符串則默認為normal
另外還可以傳數字加回調函數(動畫結束後執行的函數) 如下：
$('div').show(500,function(){ console.log('動畫結束了～') })
但最常用的是不傳任何參數(因為他的效果不好看)
* 2.slideDown/slideUp方法 不傳參數默認速度normal
若傳參數 可傳入的參數以及觸發效果則同show/hide方法一樣
且JQuery還另外封裝了slideToggle() 可直接動畫切換顯示/隱藏
* 3.fadeIn/fadeOut方法 同slideDown/slideUp的使用方法
且JQuery還一樣封裝了fadeToggle() 可直接動畫切換顯示/隱藏

JQuery 自定義動畫和停止動畫

若我們給滑鼠移入移出事件添加動畫效果
會發現當鼠標反覆移入移出時 會造成動畫跑不完的情況
其實是因為JQuery會把所有動畫存在動畫隊列中逐個執行
為避免跑不完的情況發生 在執行動畫前先使用stop方法停止動畫

* 1.自定義動畫animate方法 此方法有四個參數
參數一:對象(需要的動畫樣式) 唯一必填的
參數二:動畫執行的速度 不填默認normal(400ms)
參數三:動畫的執行效果 swing(鞦韆)/linear(勻速) 默認是swing
參數四:回調函數(動畫執行完畢後執行的函數)
* 2.停止當前動畫stop方法 此方法有兩個參數 通常不傳參
參數一:clearQueue(清除動畫隊列) 傳入布爾值
參數二:jumpToEnd(跳轉至目前停止動畫的最終效果) 傳入布爾值
當stop不傳參時 默認兩個參數都是false
